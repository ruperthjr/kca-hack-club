### 2. CS50: Introduction to Computer Science (Harvard)
**Platform:** edX  
**Cost:** Free to audit  
**Certificate:** $199 (optional)

**The Most Famous CS Course**

**What You'll Learn:**
- C programming
- Algorithms and data structures
- Memory management
- Python
- SQL databases
- Web development (HTML, CSS, JS, Flask)
- Final project

**Workload:** 10-20 hours/week for 12 weeks

**Best For:** Everyone (foundational)

**Units:** Fundamentals of Computer Systems, All programming units

**Certificate Value:** Highly recognized

**Link:** https://cs50.harvard.edu/x/

**Related Courses:**
- CS50P: Introduction to Programming with Python
- CS50W: Web Programming
- CS50 AI: Artificial Intelligence
- CS50 Mobile: Mobile App Development

**Pro Tip:** Start with CS50P if new to programming

**Partnerships:** Yale, Stanford, Google, IBM, Meta

**Top Specializations:**

- **Duration:** 6 months

- **Best For:** Everyone (career backup)

## Course Completion Strategies

### Before Starting
1. **Review syllabus** completely
2. **Plan time** in calendar
3. **Join study group**
4. **Set completion date**

### During Course
1. **Take notes** (Notion/Obsidian)
2. **Code along** don't just watch
3. **Do all exercises**
4. **Ask questions** in forums
5. **Weekly review** of progress

### After Completion
1. **Build project** using skills
2. **Add to portfolio**
3. **Write blog post** about learnings
4. **Help others** in forums
5. **Review notes** monthly
