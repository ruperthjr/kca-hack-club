**Sunday:** Review & practice

### Intermediate Level (Month 4-6)

**Daily:** Selective learning + project work

##  How to Use YouTube Effectively

### The Right Way
1. **Code Along:** Don't just watch
2. **Take Notes:** Use Notion/Obsidian
3. **Build Projects:** Apply immediately
4. **Speed Control:** 1.5x-2x for reviews
5. **Pause & Think:** Before solutions revealed

### Avoid Tutorial Hell
-  Watching without doing
-  Course hopping
-  Not building own projects
-  One course at a time
-  Build portfolio projects
-  Apply to real problems

##  Video Quality Indicators

**High Quality:**
-  Clear audio
-  Updated content (2022+)
-  Structured curriculum
-  Source code provided
-  Active community

**Skip:**
-  Poor audio/video
-  Outdated (pre-2020 for web)
-  Clickbait without substance
-  No follow-along code
