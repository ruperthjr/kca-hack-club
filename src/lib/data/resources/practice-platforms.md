# Coding Practice & Challenge Platforms

##  Essential Platforms

### LeetCode
**Free:**  | **Premium:** $35/month (optional)

The gold standard for technical interview preparation.

**Best For:** Everyone preparing for internships/jobs

**Difficulty Levels:**
- Easy: Build fundamentals
- Medium: Most interview questions
- Hard: Advanced problem-solving

**Recommended Study Paths:**
- **Blind 75:** Most important interview problems
- **NeetCode 150:** Expanded essential problems
- **Grind 75:** Structured 8-week plan

**Key Features:**
- 2500+ problems
- Company-tagged questions (Google, Meta, Amazon)
- Built-in code editor with 20+ languages
- Discuss section with solutions
- Mock interviews

**Link:** https://leetcode.com/

**Pro Tips:**
- Start with Easy problems
- Focus on patterns, not memorization
- Use NeetCode's free video explanations
- Track progress in our shared spreadsheet

### Advanced (12+ months)
**Daily:** 1-2 Medium-Hard problems (90-120 min)  
**Weekly:** 12-20 problems + 1-2 contests

##  Study Strategies

### Pattern Recognition
Focus on these core patterns:
1. Two Pointers
2. Sliding Window
3. Binary Search
4. Depth-First Search (DFS)
5. Breadth-First Search (BFS)
6. Dynamic Programming
7. Backtracking
8. Greedy Algorithms

### Problem-Solving Framework
1. **Understand:** Read problem 2-3 times
2. **Examples:** Work through examples manually
3. **Brute Force:** Think of simplest solution
4. **Optimize:** Improve time/space complexity
5. **Code:** Implement clean solution
6. **Test:** Run through edge cases
7. **Review:** Study optimal solutions

## Pro Tips

1. **Quality > Quantity:** Understand deeply rather than solving many superficially
2. **Spaced Repetition:** Revisit problems after 1 week, 1 month
3. **Time Yourself:** Practice under interview conditions
4. **Explain Solutions:** Teach others to solidify understanding
5. **Focus on Weak Areas:** Don't just solve what you know
6. **Consistency Beats Intensity:** Daily practice > occasional marathons
