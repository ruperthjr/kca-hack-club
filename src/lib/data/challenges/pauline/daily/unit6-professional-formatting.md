# Professional Word Document Formatting Challenge

## Overview

Professional document formatting is a critical skill for business, academic, and technical communication. This challenge will guide you through creating a business-ready "Quarterly Project Report" in Microsoft Word, focusing on advanced use of styles, tables, and lists to ensure clarity, consistency, and professionalism.

## Objective

Create a professionally formatted Word document featuring three heading levels, a formatted financial table, and a multilevel bulleted list, all styled according to business standards.

## Prerequisites

- Microsoft Word (2019, 2021, or Office 365)
- Basic knowledge of Word interface and navigation
- Sample project data for report sections and table
- Ability to capture and annotate screenshots

## Instructions

### Part 1: Document Setup & Heading Styles

1. Set page margins to 1" (all sides), orientation to Portrait, and paper size to Letter (8.5" × 11").
2. Insert a centered header with the document title and a centered footer with page numbers (Page X of Y).
3. Set document properties: Title ("Quarterly Project Report"), Author (your name), Company ("Tech Solutions Inc."), Date (current date).
4. Create the required heading structure using Heading 1, 2, and 3 styles. Modify styles for font, size, color, alignment, and spacing as specified:
     - Heading 1: Calibri Light, 16pt, Bold, Dark Blue, Centered
     - Heading 2: Calibri, 14pt, Bold, Dark Blue, Left
     - Heading 3: Calibri, 12pt, Bold, Blue, Left

### Part 2: Table Creation & Formatting

1. After the "Financial Overview" section, insert a table titled "Q1 Project Budget vs Actual" with 7 rows (including header) and 4 columns.
2. Enter the provided budget data.
3. Format the table:
     - Header: Calibri 11pt Bold, Dark Blue background, White text, centered, 1pt borders
     - Data rows: Calibri 11pt, alternating white/light gray, left-align Category, right-align numbers, bottom border 0.5pt gray
     - Total row: Calibri 11pt Bold, Light Blue background, double top border 1pt Dark Blue
     - Numbers: Currency format, negatives in red with parentheses, positives in green, no decimals
     - Set column widths: Category 3.5", others 1.5"

### Part 3: Multilevel Bulleted List

1. After the "Next Steps" section, insert a multilevel bulleted list with at least three levels and eight items, following the provided structure.
2. Format bullets:
     - Level 1: •, 0.25" indent, Calibri 11pt Bold, 6pt after
     - Level 2: ◦, 0.5" indent, Calibri 11pt, 3pt after
     - Level 3: ▪ or ─, 0.75" indent, Calibri 11pt Italic, 0pt after
     - Line spacing: 1.15, hanging indent

## Tips & Common Mistakes

- Always use and modify built-in styles for headings and lists
- Avoid manual tabs or inconsistent bullet symbols
- Double-check table formatting and number styles
- Ensure all spacing and colors match specifications
- Don’t merge table cells or use heavy borders
