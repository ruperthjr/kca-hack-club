# Microsoft Word Document Challenge

## Overview

Professional document creation is essential for business communication, reports, and academic work. Microsoft Word's formatting features ensure consistency, readability, and professional appearance.

## Objective

Create a professionally formatted Word document about "Quarterly Project Report" that includes three heading levels, a formatted table with data, and a bulleted list with sub-items. Apply consistent styles throughout.

## Instructions

### 1. Document Setup and Headings

**Document Properties:**  
- Title: Quarterly Project Report  
- Author: Your Name  
- Company: Tech Solutions Inc.  
- Date: Current Date

**Heading Structure:**  
- Level 1: Document Title (Quarterly Project Report)  
    - Style: Heading 1  
    - Font: Calibri Light, 16pt, Bold, Dark Blue  
    - Alignment: Center  
    - Spacing: 12pt after  
- Level 2: Main Sections (Executive Summary, Project Status, Financial Overview, Next Steps)  
    - Style: Heading 2  
    - Font: Calibri, 14pt, Bold, Dark Blue  
    - Alignment: Left  
    - Spacing: 6pt before, 6pt after  
- Level 3: Sub-sections (Q1 Achievements, Q2 Goals, Team Performance)  
    - Style: Heading 3  
    - Font: Calibri, 12pt, Bold, Blue  
    - Alignment: Left  
    - Spacing: 3pt before, 3pt after  

**Required Heading Structure:**  
- Quarterly Project Report (Heading 1)
    - Executive Summary (Heading 2)
        - Overview (Heading 3)
    - Project Status (Heading 2)
        - Q1 Achievements (Heading 3)
        - Q2 Goals (Heading 3)
    - Financial Overview (Heading 2)
        - Budget Allocation (Heading 3)
    - Next Steps (Heading 2)
        - Team Performance (Heading 3)

### 2. Table Creation and Formatting

**Table Requirements:**  
- Table Title: "Q1 Project Budget vs Actual"  
- Position: After "Financial Overview" section  
- Rows: 7 (including header)  
- Columns: 4  

**Table Structure:**  
| Category           | Budgeted | Actual  | Variance  |
|--------------------|----------|---------|-----------|
| Personnel          | $45,000  | $42,500 | ($2,500)  |
| Equipment          | $20,000  | $22,000 | $2,000    |
| Software Licenses  | $15,000  | $14,500 | ($500)    |
| Training           | $8,000   | $6,500  | ($1,500)  |
| Contingency        | $12,000  | $10,000 | ($2,000)  |
| **Total**          | $100,000 | $95,500 | ($4,500)  |

**Table Formatting Specifications:**  
- Header Row: Calibri 11pt Bold, Dark Blue background, White text, Centered, 1pt borders
- Data Rows: Calibri 11pt, alternating White/Light Gray (RGB: 240,240,240), Left-align Category, Right-align numbers, Bottom border 0.5pt Gray
- Total Row: Calibri 11pt Bold, Light Blue (RGB: 200,230,255) background, Double top border 1pt Dark Blue
- Number Formatting: Currency, $ symbol, negatives in red with parentheses, positives in green, 0 decimals
- Column Widths: Category 3.5", others 1.5"

### 3. Bulleted List Creation

**List Requirements:**  
- Position: After "Next Steps" section  
- Type: Multilevel bulleted list (3 levels)  
- Items: Minimum 8 bullets across all levels  

**Required List Structure:**  
- Quarterly Action Items:
    - Finalize Q2 Project Plan
        - Develop detailed timeline
            - Assign task owners
            - Set milestone dates
        - Allocate resources
    - Review Team Performance
        - Conduct 1:1 meetings
            - Discuss career goals
            - Provide constructive feedback
        - Update skill matrix
    - Prepare Client Presentation
        - Create executive summary
        - Develop demo materials
    - Optimize Budget Utilization
        - Identify cost savings
        - Reallocate unused funds

**Bullet Formatting Specifications:**  
- Level 1: •, 0.25" indent, Calibri 11pt Bold, 6pt after
- Level 2: ◦, 0.5" indent, Calibri 11pt, 3pt after
- Level 3: ▪ or ─, 0.75" indent, Calibri 11pt Italic, 0pt after
- Line spacing: 1.15, hanging indent, consistent spacing

### 4. Document-Wide Formatting

**Page Setup:**  
- Margins: 1" all sides  
- Orientation: Portrait  
- Paper Size: Letter (8.5 × 11")  
- Header: Document title (centered)  
- Footer: Page number (Page X of Y, centered)

**Font Scheme:**  
- Body: Calibri 11pt  
- Headings: Calibri Light/Bold as specified  
- Table: Calibri 11pt  
- Line Spacing: 1.15 (body), 1.0 (tables)

**Color Scheme:**  
- Primary: Dark Blue (RGB: 0,32,96)  
- Secondary: Blue (RGB: 0,112,192)  
- Accent: Light Blue (RGB: 200,230,255)  
- Text: Black (body), White (table headers)

**Styles Application:**  
- Modify built-in Heading 1/2/3 styles
- Create table and bullet list styles
- Apply styles throughout

### 5. Step-by-Step Creation Guide

**Phase 1: Document Setup**  
- Open Word, set margins, insert header/footer, add page numbers

**Phase 2: Heading Creation**  
- Type and style headings per structure and formatting specs

**Phase 3: Table Creation**  
- Insert table, enter data, format header/data/total rows, adjust widths, format numbers

**Phase 4: Bulleted List Creation**  
- Insert multilevel list, define bullets, apply formatting

### 9. Assessment Criteria

- **Headings (30%)**: Three levels, consistent, hierarchical, appropriate content
- **Table (30%)**: Correct size, formatting, alternating colors, total row, number formatting, borders
- **Bulleted List (20%)**: Three levels, 8+ items, formatting, indentation, logical structure
- **Professionalism (20%)**: Appearance, spelling/grammar, consistency, properties, file naming

### 11. Common Mistakes to Avoid

- Not using/setting styles for headings and bullets
- Inconsistent spacing or formatting
- Wrong heading hierarchy
- Merged table cells, inconsistent number formatting, heavy borders
- Manual tabs for bullets, inconsistent symbols, too many levels
